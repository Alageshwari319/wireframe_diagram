# UI-UX-EXPERIMENT4

## Aim:
To design and develop wireframe diagrams for the Flipkart mobile application screens using Balsamiq-style UI wireframing principles, including the Splash Screen, Login Screen, Home Screen, and Checkout Screen, in order to improve user experience, navigation flow, and interface usability.

## Algorithm:

### Step 1: Identify Required Screens

Selected the major screens required in the Flipkart shopping flow:

Splash Screen
Login Screen
Home Screen
Checkout Screen

### Step 2: Analyze User Flow

Studied the sequence of user interaction:

User opens the app.
User logs into the application.
User browses products on the home page.
User proceeds to checkout and places the order.

### Step 3: Create Low-Fidelity Wireframes

Designed rough wireframe layouts for each screen using Balsamiq-style structure:

Placeholder images
Basic buttons
Navigation elements
Text fields
Product cards

### Step 4: Design Splash Screen

Added:

Flipkart logo
Brand tagline
Minimal loading interface
Simple onboarding indicators

Purpose:

Establish branding and smooth app loading experience.

### Step 5: Design Login Screen

Added:

Email/mobile input field
Password field
Login button
Social login options
Account creation link

Purpose:

Enable secure and easy authentication.

### Step 6: Design Home Screen

Added:

Search bar
Navigation icons
Product categories
Deals section
Product cards
Bottom navigation menu

Purpose:

Improve product discovery and navigation.

### Step 7: Design Checkout Screen

Added:

Checkout progress bar
Delivery address section
Order summary
Pricing details
Place Order button

Purpose:

Simplify purchase completion process.

### Step 8: Evaluate Wireframe Design

Verified:

Layout consistency
Navigation clarity
Readability
User-friendly structure
Proper visual hierarchy
